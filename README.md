Proyecto Final React JS - Coderhouse

Este proyecto es una Single Page Application (SPA) de e-commerce desarrollado con React JS y Vite, como entrega final del curso de React de Coderhouse.

Tecnologías Utilizadas
- React JS + Vite
- Bootstrap
- Firebase (Firestore)
- React Router DOM
- Context API

Funcionalidades
- Catálogo de productos dinámico desde Firebase
- Detalle de producto
- Carrito de compras con Context API
- Checkout y generación de orden en Firestore
- Navegación con React Router

Instalación

```
npm install
npm run dev
```

Variables de Entorno
Asegurar de poder incluir un archivo `.env` con tus claves de Firebase.
